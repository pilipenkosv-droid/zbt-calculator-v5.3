## 🎬 Анимации

### Появление элементов
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slideDown {
  from { transform: translateY(-100%); }
  to { transform: translateY(0); }
}

@keyframes slideUp {
  from { transform: translateY(100%); }
  to { transform: translateY(0); }
}
```

### Transitions
- **Быстрые:** 0.2s (кнопки, ховеры)
- **Средние:** 0.3s (меню, аккордеоны)
- **Медленные:** 0.5s (формы, успех)
